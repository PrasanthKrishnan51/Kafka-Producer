# 📤 Kafka Producer Service

A production-ready Kafka Producer built using Spring Boot and Apache Kafka.  
This service publishes messages to Kafka topics asynchronously.

---

## 🚀 Tech Stack

- Java 17+
- Spring Boot
- Spring for Apache Kafka
- Maven
- Docker (Optional)

---

## 📌 Features

- Publish messages to Kafka topics
- Asynchronous message sending
- JSON message support
- Configurable via application.yml
- Delivery acknowledgment handling
- Retry support
